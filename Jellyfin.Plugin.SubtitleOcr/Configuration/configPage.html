Path to a Subtitle Edit-compatible .nocr file. Leave empty to use the bundled Latin database. Databases trained in Subtitle Edit's nOCR window drop in directly.
Optional overrides matched on the subtitle's language (ISO 639 code, e.g. rus, ell). The path may be absolute or a bare file name resolved against the plugin's nocr data folder.
